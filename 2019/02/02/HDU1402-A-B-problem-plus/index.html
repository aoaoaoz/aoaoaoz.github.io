HDU1402 A*B problem plus

发表于 2019-02-02

题意描述

Calculate A * B.

Input

Each line will contain two integers A and B. Process to end of file.

Note: the length of each integer will not exceed 50000.

Output

For each case, output A * B in one line.

Sample Input

1 2
1000 2

Sample Output

2
2000

题意：

求A*B，A，B的长度小于50000。

思路：

FFT模板题。
